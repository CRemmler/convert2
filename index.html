Converter 2.0

Convert "NetLogo" to "NetLogo Hubnet/GbCC Web"

1. Find a NetLogo file or NetLogo Hubnet file.

Run NetLogo 6.x.

Open a NetLogo file like "Fireworks" or a NetLogo Hubnet file like "Gridlock Hubnet".

Save file with a .nlogo extension.

Choose Layout:
Disease Layout (Default Layout for NetLogo Hubnet file)
Introbuttons Layout (Default Layout for NetLogo file)
Segregation Layout
Fishtank Layout
Newtonia Layout
Fireworks Layout
Custom Layout

Layouts can include the following settings:
Allow students to have Command Center, NetLogo Code, Gallery and Model Info Tabs.

Allow canvases to have multiple layers.
Allow students to select multiple canvases.
Allow canvases to act as forever buttons.
Allow students the option to listen or ignore.

2. Upload a NetLogo file or a NetLogo Hubnet file.

3. Receive Download of a NetLogo Hubnet/GbCC Web zip file.

4. Run NetLogo Hubnet/GbCC Web App locally.
Install node and npm.
Unzip the compressed NetLogo Hubnet Web file.
In a command line, navigate to the unzipped folder.
Type npm install to install dependencies.
This reads the package.json file and installs dependencies in a node-modules folder.
Type node server.js to run the server (use ctrl-c to stop it)
Open a browser and go to localhost:3000

5. Troubleshoot
If there is an error like There is already a global variable called STEP-SIZE, then comment out that global variable.

6. Host NetLogo Hubnet Web App remotely.
Build and deploy on Heroku.com
